<h3 align="center">1. Hub starten</h3>

```bash
docker run -d --name myiotgrid \
  -p 5000:5000 -p 1883:1883 \
  ghcr.io/myiotgrid/hub:latest
```

<br>

<h3 align="center">2. Sensor verbinden</h3>

```bash
cd grid-sensor && pio run --target upload
```
